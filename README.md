# 📸 LYA Fotografía

Sitio web responsive desarrollado para un estudio de fotografía, diseñado para mostrar de forma elegante los servicios, galerías y la identidad visual de la marca.

## 🌐 Demo

🔗 https://lyafotografia.netlify.app/pages/inicio.html

## ✨ Características

- Diseño moderno y minimalista.
- Interfaz totalmente responsive.
- Galería fotográfica con una presentación visual atractiva.
- Navegación intuitiva entre las distintas secciones.
- Organización modular de los estilos mediante Sass.
- Animaciones y transiciones para mejorar la experiencia del usuario.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS
- Sass (SCSS)
- Flexbox
- CSS Grid
- Git & GitHub
- Netlify (Deploy)

## 📁 Estructura del proyecto

```
├── css/
├── scss/

├── pages/
├── index.html
└── README.md
```

## 🎯 Objetivo

El propósito de este proyecto es aplicar buenas prácticas de desarrollo mediante la creación de un sitio web profesional para un estudio fotográfico, implementando una estructura organizada, estilos reutilizables con Sass y un diseño adaptable a diferentes dispositivos.

## 📱 Responsive Design

El sitio fue desarrollado siguiendo el enfoque **Mobile First**, garantizando una correcta visualización en:

- 📱 Celulares
- 📲 Tablets
- 💻 Computadoras
